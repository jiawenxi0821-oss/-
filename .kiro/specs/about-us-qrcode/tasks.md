# 实现计划：关于我们页面二维码生成工具

## 概述

本实现计划将创建一个 Python 命令行工具，用于生成包含"关于我们"页面 URL 的二维码图片。系统包括核心二维码生成功能、配置文件管理、命令行接口、GitHub Pages 自动部署配置，以及完整的验证和测试功能。

## 下一阶段执行优先级（继续设计结论）

为尽快打通“公网 URL -> 可扫码二维码”的完整业务链路，建议按以下顺序推进：

- [x] A. 先完成部署链路（需求 5）
  - 新增 `.github/workflows/deploy-pages.yml`，发布 `ui copy` 到 GitHub Pages
  - 产出固定访问地址：`https://<username>.github.io/<repo>/about.html`
- [x] B. 再完成配置与 CLI（需求 6, 7）
  - 新增 `config.json` + `qrcode_config.py`
  - 新增 `qrcode_cli.py` 支持 `generate / from-config / batch / verify`
- [x] C. 最后完成验证与批量（需求 8, 9, 10）
  - 新增 `verify_deployment.py`（URL 可达 + 二维码解码一致性）
  - 新增 `batch_generate.py`（失败隔离 + 成功失败统计）
  - 增加性能基线测试（生成耗时、文件体积、内存）
